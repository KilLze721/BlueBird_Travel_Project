# 青鸟旅行网站  

模拟“携程/途牛”核心业务，打造一个高颜值的旅游线路展示与预订系统。 

核心功能与页面模块 

**1.** **用户认证与权限系统** 

➢ 登录功能：用户名/密码验证；登录成功后，前端维持用户状态，并展示用户昵称。 

**2.** **旅游线路可视化浏览与筛选** 

➢ 线路展示：线路列表以图文卡片布局展示（一行4列），每张卡片包含：封面图、线路名称、价格、等 

级等信息； 

➢ 数据获取：所有线路数据均通过接口从数据库读取（线路信息需预先导入）； 

➢ 筛选查询：支持按线路名称（模糊）、线路类型（下拉框）进行组合筛选。 

**3.** **线路预订与订单创建** 

➢ 预订操作：点击卡片上的“立即预订”按钮，弹出预订窗口； 

➢ 订单创建：用户填写游玩日期、预订人数； 

➢ 提交后，后端必须将当前登录用户的 ID、线路 ID 和订单信息写入订单表； 

➢ 反馈：下单成功后，提示用户“预订成功，请前往我的订单查看”。 

**4.** **我的订单中心** 

➢ 订单列表：用户登录后，能查询到自己创建的订单记录。 

➢ 退单操作：点击“取消订单”按钮，弹出确认提示，确认后将订单状态更新为“己取消”。
